
# Sound Assets

To enable audio haptics, add these sound files to this folder:

## Required Files

1. **hover.mp3** - Soft swoosh/brush sound for hover states (~50ms)
2. **click.mp3** - Subtle click/tink sound for button clicks (~100ms)

## Recommendations

- Use short, organic sounds (water drops, glass tap, marble contact)
- Keep files under 50KB each
- Recommended format: MP3 or WAV
- Volume will be automatically set to 20%

## Sources

You can find free UI sounds at:
- https://freesound.org
- https://soundbible.com
- https://mixkit.co/free-sound-effects/

## Usage

The sounds will play automatically when:
- Hovering over buttons (InteractiveButton)
- Clicking buttons
- Hovering over gallery items

Use the mute toggle (speaker icon) in the bottom-right corner to disable sounds.
